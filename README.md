# blockchainJS
Create a simple JavaScript blockchain.

## Blockchain setup
Link packages first by typing this command in terminal:
```
yarn install
```

## Steps to create one
 1. Blocks
 2. Blockchain
 3. API
 4. Peer Server
 5. Proof of Work
 6. Wallet
 7. Transaction Pool
 8. Miner
